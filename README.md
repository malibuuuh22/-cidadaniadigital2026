# 🌐 Cidadania Digital e IA: O Impacto das Deepfakes e da Desinformação na Sociedade

Este é um projeto de conscientização digital desenvolvido em formato de página web única (*Single Page Application*). O objetivo principal é educar e engajar a comunidade escolar (alunos, professores e responsáveis) sobre o uso ético da tecnologia e a segurança no ambiente virtual.

---

## 🎯 Objetivo do Projeto

Com o avanço das ferramentas de Inteligência Artificial Generativa, o compartilhamento de mídias manipuladas e notícias falsas tornou-se um desafio crítico. Este site serve como uma ferramenta pedagógica para:
* **Conscientizar** sobre o conceito de cidadania digital e responsabilidade na rede.
* **Ensinar** a identificar *deepfakes* (vídeos, áudios e imagens adulteradas por IA).
* **Alertar** sobre os perigos da automação de *fake news*.
* **Promover** boas práticas de segurança e proteção de dados pessoais na internet.

---

## 🚀 Funcionalidades do Site

| Funcionalidade | Descrição | Tecnologia Principal |
| :--- | :--- | :--- |
| **Design Responsivo** | Adapta-se perfeitamente a computadores, tablets e celulares. | CSS3 / Media Queries |
| **Navegação Inteligente** | Menu fixo (*sticky navigation*) que destaca a seção atual conforme a rolagem. | JavaScript (Scroll Event) |
| **Alertas Visuais** | Boxes destacados para chamar atenção sobre perigos e cyberbullying. | HTML5 / CSS3 |
| **Quiz Interativo** | O "Desafio Detetive Digital" testa o aprendizado dos alunos dinamicamente. | JavaScript (DOM Manipulation) |
